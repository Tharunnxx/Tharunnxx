## 🚀 Featured Projects

### 🚖 Tripzo — AI-Powered Ride Booking Platform
> Full-stack ride booking app with transparent ML-powered fare prediction

- Built **3 independent services**: React Native app, Node.js backend, Python ML microservice
- ML model predicts fare using **7 real-world inputs** — distance, traffic, weather, surge, vehicle type, time
- Shows **complete transparent fare breakdown** to users — every rupee explained
- Integrated **Razorpay payment gateway** with cryptographic signature verification
- Used **4 Google Maps APIs** — Directions, Distance Matrix, Places, Maps SDK

**Tech:** React Native · TypeScript · Node.js · Python · FastAPI · Scikit-Learn · PostgreSQL · Razorpay
